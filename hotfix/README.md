# The Hotfix

> 📍 Late Game · 14 decisions made

Bug in production. Users affected. The fix is one line. You can see it clearly.

But the process takes 4 hours. PR → Review → Staging → Canary → Production.

Your manager: "Can you just push it directly?"

*The Mars Climate Orbiter (1999): one team used metric units, another used imperial. The spacecraft disintegrated in Mars' atmosphere. The fix was trivial — a unit conversion. But there was no process to catch it before deployment. $327 million gone. The lesson: process exists because the cost of skipping it is always higher than the cost of following it. Eventually.*
